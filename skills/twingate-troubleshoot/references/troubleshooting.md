# Twingate Troubleshooting

## Page Title
Troubleshooting Overview

## Summary
This page serves as a high-level index for Twingate troubleshooting resources, directing users to the Help Center knowledge base, log collection guides, and community support channels. It does not contain inline troubleshooting steps but links to dedicated resources for each topic.

## Key Information
- Primary troubleshooting resource: [Twingate Help Center](https://help.twingate.com)
- **Self Service Troubleshooting Guide** is the recommended starting point for common issues
- Log collection is documented separately for Client and Connector components
- Community support available via official Twingate Subreddit
- Free enablement content is publicly available

## Prerequisites
- Access to Twingate Help Center
- Ability to collect logs from Client or Connector depending on issue type

## Step-by-Step (General Approach)
1. Start with the **Self Service Troubleshooting Guide** in the Help Center
2. Review relevant logs:
   - **Client logs** for end-user/device connectivity issues
   - **Connector logs** for network/resource access issues
3. Search Twingate Docs and Help Center for specific error or behavior
4. Post to the official Twingate Subreddit if unresolved
