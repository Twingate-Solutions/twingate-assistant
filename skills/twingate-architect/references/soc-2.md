# SOC 2 Report

## Page Title
SOC 2 Report

## Summary
Twingate maintains a SOC 2 Type 2 report, an independent third-party audit of security controls tested over a period of time. Existing customers can request access to the report via Twingate's trust portal.

## Key Information
- SOC 2 Type 2 = controls tested over a time period (not a point-in-time snapshot)
- Report covers Trust Services Criteria (security + optional additional dimensions)
- Produced by independent third-party auditor per AICPA frameworks
- Access to the report expires after **30 days**
- Twingate team manually reviews and approves access requests

## Prerequisites
- Must be an existing Twingate customer to request the report directly
- Prospective customers must contact the sales/customer team

## How to Access the SOC 2 Report

1. **Existing customers**: Visit [trust.twingate.com](https://trust.twingate.com) and submit an access request
2. **Prospective customers**: Contact Twingate's customer team directly
3. Wait for Twingate team to review and grant access
4. Access report within the 30-day window before expiration

## Gotchas
- Access expires after 30 days; you must re-request if needed after expiration
- Access is not self-serve — requests require manual approval from Twingate's team
- No direct download link; gated through trust portal
