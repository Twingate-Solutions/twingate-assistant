## ChromeOS Client

Install and setup guide for the Twingate ChromeOS Client. Installed from the Google Play Store using the Android app; same setup process as Android.

**Key Information:**
- Installed from Google Play Store (same Android app used on Android devices)
- After connecting, only traffic for configured Resources is intercepted; regular internet traffic is unaffected

**Step-by-Step:**
1. Install Twingate from Google Play Store on ChromeOS
2. Open app -- enter Twingate Network name
3. Tap "Connect" -- authenticate via IdP in browser window
4. Browser closes; Client shows online
