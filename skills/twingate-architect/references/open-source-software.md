# Twingate Open Source Software

## Summary
Twingate publishes source code for certain software components under open source licenses. Currently, the tap-windows6 driver is available under GPLv2. This page serves as the official distribution point for GPL compliance.

## Key Information
- **tap-windows6**: Windows TAP network driver, licensed under GPLv2
- Source code download available directly from Twingate's documentation site
- GPL compliance page — source must be made available alongside binary distribution

## Configuration Values
None applicable — this is a source code distribution page only.

## Gotchas
- Only one component currently listed; other Twingate components are proprietary
- tap-windows6 is a GPLv2 (not GPLv3) licensed component — copyleft obligations apply if redistributing modified versions
- No build instructions provided on this page; refer to upstream tap-windows6 documentation for compilation steps
