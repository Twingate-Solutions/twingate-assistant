# Twingate Open Source Software

## Summary
Twingate provides source code downloads for components used in their software that are distributed under open source licenses. Currently, one open source component is listed: a fork of the tap-windows6 driver under GPLv2.

## Key Information
- Twingate publishes source code to comply with open source license obligations
- Currently one listed component: **tap-windows6** (GPLv2 licensed)
- tap-windows6 is a Windows TAP virtual network driver, used for VPN-style network tunneling on Windows clients
