# Twingate DORA Locations

## Page Title
Location List (DORA)

## Summary
This page documents Twingate's service locations for DORA (Digital Operational Resilience Act) compliance. It identifies where Regulated Services are provided and where Customer Data is processed and stored. Currently, only Twingate's Private Access service qualifies as a Regulated Service.

## Key Information
- **Regulated Service**: Private Access only (no other Twingate services currently qualify)
- **Primary location**: USA — covers data processing, data storage, and service provision
- **Additional location categories listed**:
  - Countries where subprocessors are located
  - Countries where Relays are located
- Page is maintained and updated periodically (last updated ~29 days prior to retrieval)

## Prerequisites
- Relevant for customers subject to EU DORA regulation (primarily financial entities and ICT third-party service providers)
- Applies to contractual/compliance review processes, not technical implementation

## Configuration Values
_None — this is a compliance reference page with no CLI flags, API parameters, or environment variables._

## Gotchas
- The full subprocessor country list and relay country list are referenced but not fully reproduced in the retrieved content — check the live page for the complete lists
- Only Private Access is a Regulated Service; other Twingate products/services are **not** covered under this DORA location disclosure
- Locations may change; the page carries a "last updated" timestamp — verify currency before using in compliance filings
