## Android Client

Install and setup guide for the Twingate Android Client. Minimum Android version 10 required; installed from Google Play Store; authenticates via configured IdP.

**Key Information:**
- Minimum supported Android version: 10
- Install from Google Play Store (search "Twingate") or get.twingate.com
- After connecting, only traffic for configured Resources is intercepted; regular internet traffic is unaffected

**Step-by-Step:**
1. Install Twingate from Google Play Store
2. Open app -- enter Twingate Network name
3. Tap "Sign in to connect" -- authenticate via IdP in browser window
4. Browser closes; Client shows online
