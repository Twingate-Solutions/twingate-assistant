## Gotchas
- **Redistribution requirements**: Apache 2.0 and BSD 3-Clause components require license text inclusion in distributed works
- **BSD 3-Clause**: Prohibits using contributor names for endorsement without permission
- **Apache 2.0**: Patent license termination clause triggers if litigation is filed against contributors
- **fmt**: Includes an exception allowing embedded object code redistribution without copyright notices — unique among the listed licenses
