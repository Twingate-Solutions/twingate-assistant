## Team

Landing page for the Team section of the Twingate Admin Console. Links to Users, Groups, and Identity Provider configuration.

**Key Information:**
- Users: principals that Twingate grants Resource access to; added manually or synced via IdP
- Groups: determine which Resources a User can access; all Users in a Group can access the Group's Resources
- Identity Providers: integrate for user authentication and automatic group membership synchronization
