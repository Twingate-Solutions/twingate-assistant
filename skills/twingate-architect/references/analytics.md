## Analytics

Landing page for Twingate's reporting and analytics capabilities. Links to three sub-areas: network traffic analysis, audit log export, and user activity reporting.

**Available Reports:**
- **Network Traffic** -- per-connection logs between Clients and Connectors; viewable in Admin Console, exportable as CSV, streamable to S3, or output in real-time from the Connector process
- **Audit Logs** -- Admin action log (create/edit/delete events); exportable for the lifetime of the account
- **Usage Reporting** -- user activity report showing last access time per user; exportable from Admin Console
