## Upgrade to Twingate Home

Step-by-step guide to redeeming a Twingate Home promo code and upgrading from a Starter plan. Upgrade is immediate upon code redemption.

**Prerequisites:**
- Active Starter plan account
- Twingate Home access code (obtain from the Twingate Access Pass page if not already held)

**Step-by-Step:**
1. Sign into the Admin Console
2. Settings → Manage Plan
3. Locate "Redeem Twingate Home Promo Code" section under the current plan
4. Click the section to expand it
5. Enter the Twingate Home access code in the promo code field
6. Click Apply Code
7. Confirm success: green banner "Promo code applied successfully!" appears

**Key Information:**
- Upgrade takes effect immediately upon redemption
- All Twingate Home features are unlocked at that point
