# Upgrade Starter to Twingate Home

## Summary
Describes how to redeem a promo code to upgrade a Twingate Starter plan account to Twingate Home. The upgrade takes effect immediately upon successful code redemption.

## Key Information
- Upgrade applies instantly — no waiting period
- Requires an existing Starter plan account with Admin Console access
- Success indicated by green banner: "Promo code applied successfully!"

## Prerequisites
- Active Twingate Starter plan account
- Admin Console access
- Twingate Home access code (obtain from [Twingate Access Pass](https://www.twingate.com/access-pass) if not already acquired)

## Step-by-Step

1. Sign into the Admin Console
2. Click **Settings**
3. Click **Manage Plan** (navigates to billing tab)
4. Locate the **Redeem Twingate Home Promo Code** section under current plan
5. Click to expand the **Redeem Twingate Home Promo Code** section
6. Enter your access code in the promo code field
7. Click **Apply Code**
8. Confirm success via green banner at top of page

## Gotchas
- Must be on Starter plan to see the redemption section; other plan tiers may not show this option
- No mention of code expiry, but obtain code before attempting upgrade
