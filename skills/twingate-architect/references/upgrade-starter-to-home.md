# Upgrade Starter to Twingate Home

## Summary
Describes how to redeem a promo code to upgrade a Twingate Starter account to Twingate Home. The upgrade takes effect immediately upon successful code redemption via the Admin Console.

## Key Information
- Upgrade is instant — no waiting period after code is applied
- Success is confirmed by a green banner: "Promo code applied successfully!"
- Access codes are obtained from the Twingate Access Pass page (separate from this flow)

## Prerequisites
- Existing Twingate Starter plan account
- Admin Console access
- Valid Twingate Home promo/access code

## Step-by-Step

1. Sign into the Admin Console
2. Click **Settings**
3. Click **Manage Plan** (navigates to billing tab)
4. Locate **Redeem Twingate Home Promo Code** section under current plan
5. Click on the section to expand it
6. Enter the access code in the promo code field
7. Click **Apply Code**

## Gotchas
- If you don't have an access code, you must first visit the Twingate Access Pass page — codes are not issued during this flow
- No configuration values, CLI flags, or API parameters involved; this is UI-only
