# Upgrade Starter to Twingate Home

## Summary
Guides users through redeeming a promo code to upgrade a Starter plan account to Twingate Home. The upgrade takes effect immediately upon successful code redemption.

## Key Information
- Upgrade is instant — no delay after code redemption
- Requires an existing Starter plan account
- Access codes obtained separately via the Twingate Access Pass page

## Prerequisites
- Active Twingate Starter plan account
- Admin Console access
- Valid Twingate Home promo/access code (obtain at twingate.com/access-pass if needed)

## Step-by-Step

1. Sign into the **Admin Console**
2. Click **Settings**
3. Click **Manage Plan** (navigates to billing tab)
4. Locate **Redeem Twingate Home Promo Code** section under current plan
5. Click the **Redeem Twingate Home Promo Code** section to expand
6. Enter access code in the promo code field
7. Click **Apply Code**
8. Confirm success via green banner: *"Promo code applied successfully!"*

## Configuration Values
- None (UI-only flow, no CLI/API/env vars)

## Gotchas
- No access code = cannot upgrade; must visit Access Pass page first
- No rollback steps documented — treat upgrade as one-way
- Success confirmation is a green banner; absence of banner indicates the code was not applied
