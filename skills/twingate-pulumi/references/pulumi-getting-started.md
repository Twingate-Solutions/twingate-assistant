# Getting Started with Pulumi and Twingate

## Summary
Twingate supports Infrastructure as Code deployment via Pulumi, enabling automated management of Twingate environments. Guides are available for the three major cloud providers (GCP, AWS, Azure).

## Key Information
- Pulumi is the supported IaC platform for automating Twingate deployments
- Three cloud provider deployment guides available: GCP, AWS, Azure
- Uses declarative code approach for infrastructure management

## Prerequisites
- Node.js installed
- Pulumi installed and connected to a Pulumi account

## Related Docs
- Twingate + Pulumi on GCP
- Twingate + Pulumi on AWS
- Twingate + Pulumi on Azure

## Gotchas
- All guides assume Node.js as the runtime — no mention of other Pulumi language SDKs (Python, Go, .NET)
- Must have Pulumi account connected before starting any deployment guide
