# Getting Started with Pulumi and Twingate

## Summary
Twingate supports Infrastructure as Code deployment via Pulumi, allowing automation of entire Twingate environments. Guides are available for the three major cloud providers (GCP, AWS, Azure).

## Key Information
- Pulumi is the supported IaC platform for Twingate automation
- Twingate environments can be fully automated via declarative code
- Three cloud provider deployment guides available: GCP, AWS, Azure

## Prerequisites
- Node.js installed
- Pulumi installed and connected to your account
