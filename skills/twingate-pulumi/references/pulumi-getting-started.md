# Getting Started with Pulumi and Twingate

## Summary
Twingate supports Infrastructure as Code deployment via Pulumi, allowing automated management of Twingate environments. Guides exist for deploying to the three major cloud providers (GCP, AWS, Azure).

## Key Information
- Pulumi can automate entire Twingate environment configuration
- Supported cloud providers: GCP, AWS, Azure
- Implementation language: Node.js (based on prerequisites)

## Prerequisites
- Node.js installed
- Pulumi installed and connected to your account
